Netflix-Inspired Portfolio Deployment

This is a static deployment of the Netflix-inspired portfolio website for GitHub Pages. The full functionality, including the DeepSeek API integration for the chatbot, requires a Flask backend deployed on a server that supports Python applications.

Deployment Status

The website has been deployed to GitHub Pages at: https://pkowadkar.github.io

Limitations of Static Deployment

  • The chatbot is in demo mode with pre-defined responses
  • API endpoints are simulated with static responses
  • Placeholder images are used instead of actual content

For Full Functionality

To enable full functionality with the DeepSeek API integration:

  1. Deploy the Flask backend on a server that supports Python applications
  2. Set up the DeepSeek API key in the .env file
  3. Update the frontend to communicate with your deployed backend

Documentation

For complete setup and usage instructions, refer to the documentation file included in the repository.
